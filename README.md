# text2img 工具

实现将 AI （例如：ChatGPT、Gemini、Grok等AI）生成的文本转换为图片，既不违反 linux.do 论坛的 AIGC 政策 同时也不用费劲的截长图，美观好看。支持 Markdown 语法也能够保留格式。
