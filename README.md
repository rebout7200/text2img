# text2img

实现将 AI （例如：ChatGPT、Gemini、Grok等AI）生成的文本转换为图片，不违反 linux.do 论坛的 AIGC 政策

同时也不用费劲的截长图，美观好看

支持 Markdown 语法也能够保留格式

支持设置背景色和主题色

以及文本。文本对齐。字体大小。等等

喜欢的话给我点个赞吧
